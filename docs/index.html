AiDocPlus

AI 驱动的跨平台文档桌面编辑器，基于 Tauri 2 + React 19 构建，支持 macOS 和 Windows

下载最新版本 查看源码

支持 macOS (Apple Silicon) · Windows (x64)

核心功能

专为文档创作者设计的全方位 AI 写作工具

🤖

AI 内容生成

流式生成文档内容，支持停止和续写。982 个提示词模板覆盖 46 个分类，附件参考让 AI 更懂上下文。支持 OpenAI 兼容 API、智谱 GLM 等 13 个 AI 提供商。

💬

AI 聊天助手

流式对话，联网搜索，10 个内置角色。在编辑文档的同时与 AI 实时交流，获取写作建议和灵感。

✏️

Markdown 编辑器

基于 CodeMirror 6 的专业 Markdown 编辑器，支持语法高亮、代码折叠、自动补全、实时预览。三面板布局：文件树 + 编辑器 + AI 面板。

📦

多格式导出

支持导出为 Markdown、HTML、DOCX、TXT、PDF 等格式。内置原生导出引擎，同时支持 Pandoc 扩展导出。

🔄

版本控制

自动版本保存，AI 生成前自动创建版本快照。支持版本预览和一键恢复，包含完整的插件数据。

📋

项目模板

20 个内置项目模板，涵盖学术论文、商务报告、技术文档、创意写作、教育教学、公文政务、通用文档 7 大类。

🏷️

文档管理

多项目管理，文档标签与收藏，按标签筛选。多标签页同时编辑，工作区状态自动保存恢复。

🛠️

资源管理器

6 个独立的资源管理器桌面应用，可视化管理角色、AI 服务商、提示词模板、项目模板、文档模板和插件。

插件生态

21 个外部插件，通过自注册 + 自动发现机制加载，零改动主程序
