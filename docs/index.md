## 五大工作区

AiDocPlus 提供**五大工作区**协同完成内容创作：

| 工作区 | 说明 |
|--------|------|
| **生成区** | Markdown 编辑器 + AI 助手，核心写作区域 |
| **内容区** | 内容生成类插件（翻译、扩写、PPT、摘要等） |
| **合并区** | 将素材、AI 生成、人工编辑等多来源内容整合为终稿 |
| **功能区** | 功能类插件（邮件发送、文档加密、格式转换等） |
| **编程区** | 集成代码编辑器 + AI 辅助编程环境 |

通过工具栏按钮或菜单 **视图** 切换工作区。

## 功能亮点

### AI 智能写作
- **AI 对话助手** — 侧边栏实时对话，支持联网搜索，13 个 AI 服务商可选
- **智能内容生成** — 流式生成，附件参考，一键润色改写
- **982 个提示词模板** — 覆盖 46 个分类，涵盖学术、商务、创意写作等场景
- **自定义提示词** — 支持创建、导入、导出个人提示词模板

### 专业编辑器
- **Markdown 编辑** — 语法高亮、代码折叠、自动补全
- **双栏布局** — 左栏素材 + 右栏 AI 生成，对比审阅
- **多标签页** — 同时编辑多个文档，独立面板状态
- **版本历史** — 每次保存自动创建快照，随时对比和恢复

### 28 个内置插件
- **内容区**（17 个）— 翻译、摘要、PPT 生成、思维导图、时间线、海报等
- **功能区**（11 个）— 邮件发送、文档加密、格式转换、水印、语音朗读等

### 更多特性
- **跨平台** — 支持 macOS 和 Windows
- **多格式导出** — Markdown、HTML、Word (.docx)、PDF、纯文本
- **文档管理** — 项目管理、标签分类、星标收藏
- **国际化** — 中文 / English
