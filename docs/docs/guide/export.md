# 导出功能

AiDocPlus 支持将文档导出为多种格式。

## 支持的格式

| 格式 | 说明 |
|------|------|
| **Markdown** (.md) | 原始 Markdown 格式 |
| **HTML** (.html) | 网页格式，保留样式 |
| **Word** (.docx) | Microsoft Word 格式 |
| **PDF** (.pdf) | 便携文档格式 |
| **纯文本** (.txt) | 去除格式的纯文本 |

## 如何导出

1. 打开要导出的文档
2. 菜单 **文件 → 导出** 或点击工具栏导出按钮
3. 选择导出格式
4. 选择保存位置

## 项目导出

支持将整个项目导出为 ZIP 压缩包，包含所有文档和附件。

- 菜单 **文件 → 导出项目**
- 选择保存位置
