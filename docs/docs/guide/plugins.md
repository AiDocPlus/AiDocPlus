# 插件系统

AiDocPlus 内置 **27 个插件**，分为内容生成类（内容区）和功能类（功能区）。

## 进入方式

- 工具栏点击 **「内容区」** 或 **「功能区」** 按钮
- 菜单 **视图 → 内容区** 或 **视图 → 功能区**

## 插件工作流程

1. 切换到内容区或功能区
2. 从左侧列表选择插件
3. 在右侧面板中操作
4. 结果可直接插入到当前文档

## 插件管理

通过 **设置 → 插件管理** 可以启用/禁用插件，调整显示顺序。
