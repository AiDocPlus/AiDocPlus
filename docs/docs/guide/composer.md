# 合并区

合并区用于将多来源内容（素材、AI 生成、人工编辑）整合为终稿。

## 进入方式

- 工具栏点击 **「合并区」** 按钮
- 菜单 **视图 → 合并区**
- 快捷键 `Cmd/Ctrl + 3`

## 功能说明

合并区提供双栏对比视图：
- **左栏**：原始内容（素材）
- **右栏**：AI 生成内容或修改后的版本

## 工具栏功能

- **全部采纳** — 将右栏所有内容替换左栏
- **逐段采纳** — 逐段审阅，选择性采纳
- **清空右栏** — 清除 AI 生成内容

## 使用场景

1. AI 生成一篇完整文章后，在合并区逐段审阅
2. 将不同来源的素材合并为一个完整文档
3. 对比修改前后的文档内容

## 与其他工作区的协作

- **生成区** → AI 生成内容 → **合并区** 审阅采纳
- **内容区** → 插件生成翻译/摘要 → **合并区** 整合
