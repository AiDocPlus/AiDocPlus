# 生成区（编辑器）

生成区是 AiDocPlus 的核心写作区域，包含 Markdown 编辑器和 AI 助手面板。

## 进入方式

- 工具栏点击 **「生成区」** 按钮
- 菜单 **视图 → 生成区**
- 快捷键 `Cmd/Ctrl + 1`

## 双栏布局

- **左栏**：Markdown 编辑器，编写素材和正文
- **右栏**：AI 生成栏，显示 AI 生成的内容

## Markdown 编辑特性

- 语法高亮（基于 CodeMirror 6）
- 代码折叠
- 自动补全
- 行号显示
- 搜索替换（`Cmd/Ctrl + F`）
- Markdown 实时预览

## 多标签页

- 同时打开多个文档，每个标签页独立
- 拖拽标签页调整顺序
- 右键标签页可关闭、关闭其他、关闭所有

## 附件管理

- 支持在文档中插入图片
- 图片自动保存到项目的 `attachments/` 目录
- 支持拖拽图片到编辑器

## 文档管理

- 左侧文件树显示项目结构
- 支持新建、重命名、删除文档
- 支持标签分类和星标收藏

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl + S` | 保存 |
| `Cmd/Ctrl + Shift + S` | 保存所有 |
| `Cmd/Ctrl + Z` | 撤销 |
| `Cmd/Ctrl + Shift + Z` | 重做 |
| `Cmd/Ctrl + F` | 搜索 |
| `Cmd/Ctrl + H` | 版本历史 |
