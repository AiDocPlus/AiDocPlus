# 编程区

编程区集成了多语言代码编辑器和运行环境，支持 AI 辅助编程。

## 进入方式

- 工具栏点击 **「编程区」** 按钮
- 菜单 **视图 → 编程区**
- 快捷键 `Cmd/Ctrl + 5`

## 界面布局

- **左侧**：文件树 + 代码编辑器
- **右侧**：AI 编程助手 + 输出面板

## 代码编辑

- 语法高亮（支持 Python、JavaScript、TypeScript 等）
- 代码折叠
- 自动缩进
- 括号匹配

## 代码执行

- 支持 Python、Node.js 脚本运行
- 输出面板实时显示运行结果
- 支持标准输入输出

## AI 辅助编程

- 在右侧 AI 面板中描述编程需求
- AI 自动生成代码并可直接插入编辑器
- 支持代码解释、优化、调试等

## 与文档的协作

- 编程区生成的代码片段可以插入到文档中
- 文档中的代码块可以在编程区中执行
