# 功能指南

AiDocPlus 提供五大工作区和丰富的功能模块：

- [生成区（编辑器）](./editor) — Markdown 编辑器、双栏布局、文档管理
- [AI 对话与生成](./ai-chat) — AI 助手、提示词模板、上下文模式
- [插件系统](./plugins) — 27 个插件的使用方法
- [合并区](./composer) — 多来源内容整合
- [编程区](./coding) — 代码编辑、执行与 AI 辅助编程
- [模板管理](./templates) — 提示词模板和文档模板
- [导出功能](./export) — 多格式导出
- [版本历史](./version-history) — 自动版本快照与恢复
