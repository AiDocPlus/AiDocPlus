# 模板管理

AiDocPlus 提供提示词模板和文档模板两大模板系统。

## 提示词模板

### 内置模板

内置 **982 个提示词模板**，覆盖 46 个分类，包括学术写作、商务文档、创意写作、技术文档等。

### 使用模板

1. 在 AI 面板中点击 **模板** 按钮
2. 浏览分类或搜索关键词
3. 点击模板自动填入输入框
4. 修改参数后发送

### 自定义模板

- 点击 **新建模板** 创建个人模板
- 支持设置模板名称、分类、内容
- 支持导入/导出模板（JSON 格式）

## 文档模板

### 内置模板

内置 **20 个文档模板**，覆盖报告、文章、邮件、会议纪要、技术文档等场景。

### 使用文档模板

1. 新建文档时选择模板
2. 模板内容自动填入编辑器
3. 在模板基础上修改和扩展

## 资源管理器

通过菜单 **工具 → 资源管理器** 打开独立的资源管理应用，可以：

- 管理提示词模板（添加、编辑、删除、分类）
- 管理文档模板
