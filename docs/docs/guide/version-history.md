# 版本历史

AiDocPlus 自动保存文档的版本历史，支持随时查看、对比和恢复。

## 工作原理

- 每次保存文档时自动创建版本快照
- 快照包含完整的文档内容
- 存储在项目本地，不依赖外部服务

## 查看版本历史

1. 快捷键 `Cmd/Ctrl + H`
2. 或菜单 **编辑 → 版本历史**
3. 左侧显示版本列表（按时间倒序）

## 版本对比

1. 在版本列表中选择一个版本
2. 右侧显示当前版本与所选版本的差异
3. 新增内容高亮显示，删除内容标红

## 恢复版本

1. 选中要恢复的版本
2. 点击 **恢复此版本**
3. 当前文档内容将被替换为所选版本

> 恢复操作本身也会创建一个新的版本快照，可以随时撤回。

## 最佳实践

- 在重大修改前手动保存一次（`Cmd/Ctrl + S`）
- 养成定期保存的习惯
- 利用版本历史进行 A/B 版本对比
