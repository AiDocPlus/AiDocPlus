# 更新日志

## v0.3.0（2026-02-26）

### 新增功能
- **编程区** — 集成多语言代码编辑器和运行环境，支持 AI 辅助编程
- **五大工作区** — 生成区、内容区、合并区、功能区、编程区并列切换
- **首次运行引导** — 4 步交互式新手引导，帮助快速上手
- **帮助菜单增强** — 新增官网、文档、反馈入口
- **关于对话框升级** — 动态版本号、多链接入口、版权信息

### 改进
- **版本号统一** — 全面统一为 0.3.0
- **国际化完善** — 覆盖所有新增 UI 文案
- **架构精简** — 移除已废弃功能，精简代码
