# 更新日志

## v0.3.0（2026-02-26）

### 新增功能
- **编程区** — 集成多语言代码编辑器和运行环境，支持 AI 辅助编程
- **五大工作区** — 生成区、内容区、合并区、功能区、编程区并列切换
- **开放 API 系统** — HTTP API Server + JSON-RPC 网关（11 个命名空间 30+ 操作）
- **MCP Server** — 23 个工具，Claude Desktop / Cursor 直接调用
- **Python SDK / JS SDK** — 零配置 Proxy 自动代理
- **编程区自动化** — 脚本运行时自动注入 API 连接参数
- **文档级 AI 服务切换** — 每个文档可独立绑定不同 AI 服务
- **首次运行引导** — 4 步交互式新手引导
- **帮助菜单增强** — 新增官网、文档、反馈入口

### 改进
- **Monorepo 架构** — 所有源码、插件、资源合并为单一仓库
- **插件增至 27 个** — 新增多个内容生成和功能执行类插件
- **提示词模板增至 1481 个** — 覆盖 53 个分类
- **版本号统一** — 全面统一为 0.3.0
- **国际化完善** — 覆盖所有新增 UI 文案
