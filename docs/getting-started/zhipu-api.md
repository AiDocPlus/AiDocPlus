# 如何获取智谱 AI API Key

本文将指导你如何在智谱开放平台注册账号并获取免费 API Key，以便在 AiDocPlus 中使用 AI 功能。

## 为什么选择智谱 AI？

- **新用户福利**：2000万免费 Tokens，足够体验所有 AI 功能
- **国产大模型**：响应快、中文理解能力强
- **免费模型**：GLM-4.5 Flash、GLM-4 Flash 等模型可免费使用

---

## 步骤一：注册账号

1. 访问智谱开放平台：[https://open.bigmodel.cn/](https://open.bigmodel.cn/)
2. 点击右上角「注册/登录」按钮
3. 选择以下方式之一注册：
   - 微信扫码（推荐，最快捷）
   - 手机验证码
   - 邮箱注册

![注册页面](./images/zhipu-register.png)

---

## 步骤二：获取 API Key

1. 登录后，点击右上角头像进入「用户中心」
2. 在左侧菜单中找到「API Keys」
3. 点击「创建新的 API Key」按钮
4. 输入 Key 名称（如：AiDocPlus）
5. 点击确认后，**立即复制保存 API Key**

> ⚠️ **重要**：API Key 仅显示一次，请务必复制保存！

![创建 API Key](./images/zhipu-api-key.png)

---

## 步骤三：配置到 AiDocPlus

1. 打开 AiDocPlus 应用
2. 点击菜单「设置」或使用快捷键打开设置面板
3. 切换到「AI」设置页
4. 点击「创建 API 服务」按钮
5. 填写以下信息：
   - **服务商**：选择「智谱 AI (通用)」
   - **API Key**：粘贴刚才复制的 API Key
   - **模型**：推荐选择「GLM-4.5 Flash（免费）」
6. 点击「测试连接」验证配置是否正确
7. 点击「保存」完成配置

![AiDocPlus 设置](./images/aidocplus-settings.png)

## 常见问题

### Q: API Key 忘记了怎么办？

可以在智谱平台的「API Keys」管理页面重新创建新的 Key。旧的 Key 仍然有效，直到你主动删除。

### Q: 免费 Tokens 用完了怎么办？

可以充值购买更多 Tokens，或者注册新账号获取新的免费额度。

### Q: 哪些模型是免费的？

- GLM-4.5 Flash（推荐）
- GLM-4 Flash
- GLM-4.5 Air（部分免费）

### Q: 为什么选择 GLM-4.5 Flash？

- 完全免费
- 响应速度快
- 效果足够满足大多数场景
